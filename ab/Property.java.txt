**Class: Property**

**Description:**
The `Property` class is an entity class that represents a property in a real estate management system. It is annotated with `@Entity` to indicate that it is a JPA entity and with `@Table(name = "properties")` to specify the name of the database table that will store the property data.

**Attributes:**

- **id**: Unique identifier for the property. It is annotated with `@Id` to indicate that it is the primary key of the entity and with `@GeneratedValue(strategy = GenerationType.AUTO)` to specify that the value will be automatically generated by the database.
- **address**: Street address of the property.
- **city**: City in which the property is located.
- **state**: State in which the property is located.
- **zipCode**: Zip code of the property's location.
- **price**: Asking price of the property.
- **numBedrooms**: Number of bedrooms in the property.
- **numBathrooms**: Number of bathrooms in the property.
- **type**: Type of property, such as "house," "condo," or "apartment."
