**Class:** `UserAccount`

**Description:**

The `UserAccount` class represents a user account entity in an e-commerce system. It contains the following attributes:

- `id`: The unique identifier of the user account.
- `username`: The username of the user account.
- `password`: The password of the user account.
- `role`: The role of the user account (e.g., admin, customer).

**Annotations:**

- `@Id`: Specifies that `id` is the primary key of the entity.
- `@GeneratedValue(strategy = GenerationType.IDENTITY)`: Specifies that the value of `id` should be auto-generated by the database.

**Getters and Setters:**

The class also includes getters and setters for each attribute. These methods allow you to access and modify the values of the attributes.

**Example Usage:**

```java
UserAccount userAccount = new UserAccount();
userAccount.setUsername("admin");
userAccount.setPassword("password");
userAccount.setRole("ADMIN");

// Save the user account to the database
userAccountRepository.save(userAccount);
```

This code creates a new `UserAccount` object, sets its attributes, and saves it to a database using a `userAccountRepository`.
